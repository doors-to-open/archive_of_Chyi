# Copyright and Link Policy

This project is an archive index. It should not host copyrighted audio, video, scans, or images unless the project has permission or a clearly valid license.

## Media

- Link to external media instead of copying it.
- Prefer official uploads and official platform pages when available.
- Clearly label unofficial or fan-uploaded media links.
- Remove or replace links when they become unavailable or problematic.

## Metadata and Facts

Metadata, citations, and factual descriptions should be written in our own words wherever possible. When quoting, keep excerpts short and cite the source.

## Album Art and Images

Album art, photographs, broadcast stills, posters, and scans may be copyrighted. The project should avoid storing these assets until a clear image policy is decided.

## Contributor Submissions

Contributors should provide sources for factual claims. Personal memories and short fan notes should be stored separately from verified archive data.

## Disclaimer

This project is a fan archive and reference index. It is not affiliated with Chyi Yu, her labels, publishers, management, or media platforms unless explicitly stated.
